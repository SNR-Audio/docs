## Welcome to SNR Audio API Documentation

Welcome to the documentation for the SNR Audio API! This API provides powerful features for Text-to-Speech (TTS) and Automatic Speech Recognition (ASR). Whether you need to convert text into lifelike speech or transcribe audio files into text, SNR Audio API has got you covered.

### Features

- **Text-to-Speech (TTS):** Convert text into high-quality speech with customizable voices.
- **Automatic Speech Recognition (ASR):** Transcribe audio files into text with high accuracy.

### Get Started

To get started with the SNR Audio API, navigate through the provided documentation sections:

- Learn about the TTS endpoint to convert text into speech.
- Explore the ASR endpoint to transcribe audio files into text.

### Authentication

All API endpoints require authentication using a Bearer token. Make sure to include your token in the request headers for authorization.
